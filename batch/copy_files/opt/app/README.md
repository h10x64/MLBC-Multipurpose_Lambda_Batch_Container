# アプリケーション配置フォルダ

処理に必要な各種アプリケーションを配置するフォルダです。  
各言語のサンプルが置いてありますので、必要に応じで利用・参照してください。

## ビルドが必要な言語についての注意

cやjavaではコンパイルなどビルドが必要になりますので、  
Dockerfileでdocker build時に同時にビルドが行われるように設定されています。

フォルダに配置しても自動ではビルドされませんのでご注意ください。  
(※詳細はDockerfileをご参照ください。)

## 各種言語用サンプルの内容

基本的にはHelloWorldですが、shやsqlで一部特殊なサンプルが含まれます。

### sh/remote_call - SSMを利用したEC2インスタンスの打鍵バッチのサンプル

`sh/remote_call` に `opt/batch/call.sh` を利用してSSMでEC2インスタンス上でコマンド打鍵するサンプルが置いてあります。  
SSMを利用すれば踏み台サーバ等を用意せずともEC2インスタンス上での打鍵が可能となりますが、  
インスタンスに `AmazonSSMManagedInstanceCore` ポリシーを持たせる必要があるなど、  
必要な事前準備がありますので、詳しくは `sh/remote_call/README.md` をご参照ください。

### sql/run_sql.sh

psqlを利用して `sql/run_sql/sql` フォルダ以下にあるSQLファイルを実行するサンプルです。
envファイルで `DATABASE_URL` 環境変数の設定が必要です。
